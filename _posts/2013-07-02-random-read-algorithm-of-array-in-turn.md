# 第二个方案
1. 对两个数组（A，B），分别进行排序。
2. 算出我要取的位置（大数组的长度-小数组的长度 的随机值int_R）
3. 依次遍历大数组，跟小数组的值，一一对比：如果相等就继续，否则随机值递减，如果最后值为0，则代表取到了要的随机值。 
    if (A[i] == B[j]) {
        ++j;
        continue;
    } else {
        if (!--int_R) {
            return A[i];
        }
    }
